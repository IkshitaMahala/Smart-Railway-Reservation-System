# Smart Railway Reservation System

A console-based Railway Reservation System built using C++.

## Features

- Add Train
- Display Trains
- Book Ticket
- Cancel Ticket
- Waiting List
- Search Passenger
- Sort Passengers
- Train Statistics

## Technologies Used

- C++
- Vector
- Queue
- Structures
- Functions

## How to Run

Compile using any C++ compiler.

## Possible Enhancements

- Save train and passenger data using file handling.
- Add a login system for administrators.
- Assign seat numbers automatically after booking.
